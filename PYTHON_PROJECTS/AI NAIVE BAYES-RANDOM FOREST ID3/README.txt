Σε όλα τα main.py επιλέγουμε 0 για default παράδειγμα
Αν επιλέξουμε 1 τότε ορίζουμε εμείς τις τιμές m,n,k,training sizes
καθώς και τα paths για τα training & test data.
Για να εκτελέσετε το αρχείο θα πρέπει να επιλέξετε 1 βάζοντας τις δικές
σας τιμές ή εναλλακτικά αλλάζετε μέσα στον κώδικα τα default paths και 
επιλέγετε 0.

PART A -> Main.py το executable. 
Algorithms.py υλοποίηση Random Forest ID3 & Bernoulli Naive Bayes 
+ εξτρα βοηθητικές συναρτήσεις για φόρτωση, 
επεξεργασία δεδομένων, υπολογισμό μετρικών.

PART B -> Main.py το executable. 
Methods.py υλοποίηση ίδιων βοηθητικών μεθόδων με το part A 
χωρίς τους δικούς μας αλγορίθμους.

Part C -> Main.py το executable.
Methods.py υλοποίηση βοηθητικών μεθόδων μόνο όσων χρειάστηκαν 
για την χρήση RNN με word embeddings.
